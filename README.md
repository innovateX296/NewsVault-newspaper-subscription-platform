# NewsVault

NewsVault is a modern and responsive newspaper subscription and competitive exam preparation platform developed using HTML, CSS, and JavaScript. The website combines a digital news portal with an ecommerce system where users can read latest articles, purchase books, and subscribe to newspaper plans.

## Features
- Responsive Design
- Daily News Articles
- ISRO & NASA News
- Political & Current Affairs Updates
- Competitive Exam Book Store
- Subscription Plans
- Search Functionality
- Attractive UI/UX
- Ecommerce Style Product Cards
- Mobile Friendly Layout

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Font Awesome

## Project Type
Frontend Web Development Project
